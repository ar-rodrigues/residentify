# Roadmap - Remaining Features & Improvements

This document outlines the remaining features and improvements needed for the project.

---

## UI/UX Improvements

- Replace automatic sidebar expand/collapse with manual toggle button that saves user preference to localStorage
- Implement dark mode throughout the application
- Add organization switcher button for mobile devices (currently only available on desktop)
- Merge Members and Invites pages into a single unified interface with clean UI/UX

## Theme System

- Create centralized theme configuration for colors and fonts
- Implement CSS custom properties (CSS variables) for theme management
- Create ThemeProvider component for dark mode state management
- Update AntdProvider to use theme configuration
- Add theme toggle button component
- Ensure theme preference persists in localStorage
